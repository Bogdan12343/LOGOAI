API Documentation

Integrate LOGOAI's AI logo generation directly into your application.

⚠️ The public API is currently available on the Agency plan only. Full documentation coming soon.

Authentication

All API requests require an API key passed in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Base URL

https://logoai-backend.vercel.app/api

Endpoints

POST/generate-logo
Generate a logo using AI based on company name, style, and industry.
GET/logos
Retrieve all logos generated by the authenticated user.
GET/profile
Get the current user's profile including credits and plan.

Example Request

{ "companyName": "Nova Tech", "style": "minimal", "industry": "technology", "colors": "blue and white" }

Example Response

{ "imageUrl": "https://..." }
